# CollectingSlackLog

# Installation

1. 仮想環境の準備 `python3 -m venv CollectingSlackLog`
2. `pip install -r requirements.txt`
3. `.env.yml` を配置
4. `.env.yml` にSlack API トークンを設定
5. python main.py

## .env.ymlの例

```yaml
SLACK_API_TOKEN: xxxx-00000000-00000000-00000000-0x0x0x0x0x
```
